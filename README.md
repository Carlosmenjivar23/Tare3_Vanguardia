# API REST - Inventario de Productos

API REST desarrollada utilizando **Node.js**, **Express**, **MySQL** y **Sequelize ORM** para administrar el inventario de productos de una tienda.

---

## Descripción

Esta API permite gestionar productos dentro de un sistema de inventario.

Cada producto contiene:

- id  
- nombre  
- descripción  
- precio  
- stock  
- categoría  
- fecha_creación  

La API permite:

Crear productos  
*Listar productos  
*Buscar por ID  
*Actualizar productos  
*Eliminar productos  

---

## Tecnologías Utilizadas

- **Node.js** : Entorno de ejecución para JavaScript  
- **Express.js** : Framework web minimalista para APIs  
- **MySQL** : Base de datos relacional  
- **Sequelize ORM** : ORM para manejo de consultas y modelos
